# CodeAlpha Email Extractor Automation

## Description
A Python automation script that extracts email addresses from a text file and saves them to another file.

## Features
- Reads data from a text file
- Uses Regular Expressions (Regex)
- Extracts valid email addresses
- Saves results to a separate file
- Displays total emails found

## Technologies Used
- Python
- Regex
- File Handling
