# CodeAlpha Email Extractor Automation

## Overview

A Python automation script that extracts email addresses from a text file using Regular Expressions (Regex) and stores them in a separate output file.

## Features

- Reads data from text file
- Extracts valid email addresses
- Uses Regular Expressions (Regex)
- Saves extracted emails to another file
- Displays extracted emails and count

## Technologies Used

- Python
- Regex (re module)
- File Handling

## How to Run

1. Add email data inside sample.txt
2. Open terminal
3. Run:

```bash
python main.py
```

